DooWapp

Personal Project - DooWapp Website

DooWapp is a free music messaging app with an amazing patent pending feature that lets users add song quotes to messages and send. When the recipient receives the message they experience a Whole new form of communication that includes those lyrics being Played with their message.

As an intern at DooWapp I was given a number of different tasks too complete however the most interesting task was to create a website for the company. The owner wanted me to create a simple static landing page where the visitors can easily find a link to download the app and get information of what the app actually does. Therefore, I decided to include a lot of JQuery functionality (such as Smooth Scrolling so the user does not have to manually scroll down the page) which will make the website more interactive.

Technologies I used:

  • HTML
  • CSS
  • JavaScript (JQuery)
  • Bootstrap
