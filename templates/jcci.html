JCCI Zion Chapel

Personal Project - JCCI

For my first personal project I decided to create a responsive static website for my mother's church (JCCI Zion Chapel) because they did not have an online presense. Therefore I decided to make a website for them which will include basic information about the church such as contact information and histroy about the church. The website was developed with in the Flask and Bootstrap framework so the website fully responsive.

Technologies I used:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Python (Flask)
