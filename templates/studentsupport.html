Student Support

302CEM Submission - Student Support

The purpose of this assignment was to participate in a development process and use the experience I gain to reflect on it. I was working in a team of 5 to create a final product containing a full-stack solution which will be assessed as a group. Students of Coventry University have a number of ways to get help with their studies including Theta, Maths and Programming Support. Therefore, we were asked to develop a tool to enhance these processes. The final product includes user authentication, live chat room, quizzes, API, etc.

Technologies I used:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Python (Flask)
  • NodeJS
  • Angular JS(1)

Coursework Grade: 60% (2:1)
