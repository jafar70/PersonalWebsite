For my 205CDE (Developing the Modern Web) module I was given the task to create a website about any chosen topic or organisation. I decided to make a website about Coventry University Student Union. The main reason I decided to create a website about the student union is because I believe that the current Coventry University Student Union could have a simpler and better designed website as it can hard to find the information you want on the current website.

The website was developed with in the Flask and Bootstrap framework so the website fully responsive. The website had to include a database so I made a web app within the website that allows the users to add a student to the database, add a grade to a student on the database and find the student on the database using the student ID.

Technologies I used:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Python (Flask)
  • SQL
