Starfighter

Personal Project - Starfighter.io

Starfighter is a recruiting company which sources the best engineering candidates in the world using Capture The Flag online programming challenges (CTFs). Their first set is Stockfighter. They observe our entire player base, discover extraordinarily talented players, and qualify them as candidates by having them talk to an actual developer about their background and aspirations. If appropriate, we introduce them to their clients as candidates.

Starfighter had a dated and non-responsive website so I decided to create a new dynamic and responsive website for Starfighter as a personal project. Since Starfighter is a recruiting company I thought it would important to give as much information as possible to the visitors so they understand how the company work and how potential users can talk to their clients. Therefore, I decided to create a Frequently Asked Questions page (FAQs) where the user can click on a question and the page will scroll down to the answer.

Technologies I used:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Python (Flask)
